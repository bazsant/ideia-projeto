Ideias de projetos

Desenvolvido para mostrar aleatoriamente uma ideia de projeto com Tema e Linguagens de Front-end e Back-end.

A ideia geral a princípio é servir de um primeiro passo e gerar um desafio caso queira desenvolver algo mas esteja sem ideias. E/Ou queira aprender uma linguagem de programação ou alguma capacitação específica.

Pode ser usado também para sortear termos específicos.
